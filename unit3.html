บทที่ 3: "ทางแยกตัดสินใจ"

ลองสวมบทบาทเป็นตู้ขายตั๋วหนังที่ต้องเช็กอายุผู้ชม!

🤖 สถานการณ์: ตรวจสอบอายุเข้าชมหนังผี

หนังเรื่องนี้ถูกจัดเรตไว้ที่ 15+ หน้าที่ของคุณคือสร้างเงื่อนไขว่า ถ้า (if) อายุถึงให้เข้าได้ ถ้าไม่ใช่อย่างนั้น (else) ให้แจ้งเตือนว่าอายุไม่ถึง

เด็ก (5 ปี) ผู้ใหญ่ (30 ปี)
✅

เงื่อนไข: if (age >= 15)

เมื่ออายุ 15 ปีขึ้นไป

ผลลัพธ์: "ยินดีด้วย คุณเข้าชมได้"
❌

เงื่อนไข: else

เมื่อเงื่อนไขข้างบนไม่เป็นจริง

ผลลัพธ์: "ขออภัย อายุของคุณไม่ถึง"

💻 โค้ดที่เกิดขึ้นเบื้องหลัง (Real-time Code)

if (age >= 15) {
printf("Welcome to the cinema!");
} else {
printf("Sorry, you are too young.");
}
💡

คำอธิบาย: คอมพิวเตอร์จะวิ่งไปตามเส้นทางที่ "เป็นจริง" เพียงเส้นเดียวเท่านั้น เมื่ออายุเปลี่ยนไป เส้นทางของกระแสข้อมูลก็จะเปลี่ยนตามทันที!
